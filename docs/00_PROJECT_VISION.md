# 00_PROJECT_VISION.md — プロジェクト目的・理念

## ミッション

世界のアンティークコイン・モダンコインを、正確で信頼できる情報とデータで、
誰もが安心して学び、収集できる環境を提供する。

## ビジョン

日本で最も信頼されるコイン情報プラットフォームになる。

## バリュー

| 価値観 | 説明 |
|---|---|
| 正確性 | 一次資料・公式データベースに基づいた情報のみを掲載する |
| 中立性 | 販売利益を目的とした情報の偏りを排除する |
| 透明性 | 情報源・根拠を明示し、不確かな情報は「確認中」と表記する |
| 長期視点 | 短期の売買トレンドより、コインの本質的価値を伝える |
| 親しみやすさ | 初心者にも分かりやすい言葉・構成で書く |

## なぜ作るのか

日本国内にはアンティークコイン・モダンコインに関する**体系的な日本語情報がほぼ存在しない**。

- 海外（PCGS・NGC・eBay）の情報は英語が前提で、初心者の参入障壁が高い
- 国内業者の情報は販売バイアスがかかりやすい
- 「博物館」のような、中立で体系的な日本語情報プラットフォームがない

この空白を MINT COIN が埋める。

## コンセプト

**「博物館」 × 「データベース」 × 「専門家」**

- 博物館：信頼できる情報を体系的に整理して展示する
- データベース：価格・グレード・発行枚数など定量データを公開する
- 専門家：コインの価値・見方を初心者にも分かりやすく解説する

ショップではなく、**情報で信頼を積み上げる**ことを優先する。

## ターゲット

### Primary（最優先）
- 30〜70代の日本在住者
- アンティークコインに興味を持ち始めた初心者
- 資産保全の手段としてコインを検討している人
- 既存のコレクターで日本語情報を探している人

### Secondary
- NGC・PCGS のスラブコインを既に持っている中級者
- eBay・海外オークションを利用している人
- 投資目的でコインを検討しているが情報が少ない人

## 事業の流れ（情報 → 信頼 → 収益）

```
情報提供（MINT COIN）
    ↓
信頼構築（正確・中立な情報の蓄積）
    ↓
無料相談（コンタクトからの問い合わせ）
    ↓
コンサルティング（有料）
    ↓
代理購入・NGC提出サポート（有料）
    ↓
販売（直販）
    ↓
リピーター・コミュニティ
```

営業しなくても仕事が来る仕組みを、情報の質で作る。

## MINT COIN の強み（他にないもの）

1. **Variant 分類データ** — 独自に構築した Series/Variant/Portrait DB との連携
2. **日本語で読めるシリーズ解説** — Una and the Lion から Morgan Dollar まで体系的に
3. **グレード・価格データの可視化** — NGC/PCGS/eBay/国内価格を横断比較
4. **中立な立場** — 販売ありきではない情報発信
