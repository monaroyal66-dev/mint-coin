# 04_CONTENT_GUIDE.md — コンテンツ・SEO ガイドライン

## コンテンツ方針

### 基本姿勢

- **正確性優先**: 造幣局公式資料・PCGS/NGC データベースを一次資料とする
- **中立性**: 価格煽り・投資勧誘的な表現を使わない
- **読みやすさ**: 専門用語は初出時に括弧で英語・解説を付ける（例：鑑定機関（グレーディングカンパニー））
- **完結性**: そのページを読めば「分かった」と思える情報量にする

### トーン

| 良い例 | 悪い例 |
|---|---|
| 「価格は時期やグレードにより異なります」 | 「今が買い時！」 |
| 「NGC MS70 は最高グレードです」 | 「NGC MS70 は必ず値上がりします」 |
| 「参考価格として eBay 落札相場を掲載します」 | 「この価格で買えばお得です」 |

## ページ別コンテンツ要件

### シリーズ詳細ページ（最重要）

`/series/{slug}/` — このページを日本一のクオリティにする。

#### 必須セクション

1. **概要ボックス**（ページ最上部）
   - 発行元（造幣局）
   - 素材・品位（例：金 22ct .9167）
   - 発行年（例：1817年〜現在）
   - 特徴（1〜2文）

2. **歴史・背景**
   - 発行の経緯
   - デザインの変遷
   - 代表的な彫刻家・肖像

3. **デザイン解説**
   - 図柄の意味・モチーフ
   - 表面（オブバース）と裏面（リバース）の説明

4. **Variant（種類）一覧**
   - サイズ別（1oz / 1/2oz / 2oz etc. または Face Value）
   - グレード別（MS/PR/SP）
   - 発行枚数（分かる範囲で）

5. **価格情報**（将来 DB 連携）
   - eBay 相場（直近落札平均）
   - 国内（ヤフオク）相場
   - グレード別価格差

6. **関連コラム・記事**

7. **関連シリーズ**

#### SEO 要件
- `title`: `{シリーズ名} | MINT COIN`
- `description`: シリーズ名・素材・特徴・発行元を含む 120 文字以内
- H1: シリーズ名（英語名 + 日本語カナ読み）
- H2 以降: 上記セクション見出し

## SEO ガイドライン

### ページ別 title・description

| ページ | title | description |
|---|---|---|
| `/` | `MINT COIN` | MINT COINは、アンティークコイン・モダンコインの基礎知識、シリーズ解説、グレーディング情報を発信する情報サイトです。 |
| `/series/` | `シリーズ解説 \| MINT COIN` | Royal Mint、Sovereign、Una and the Lion など世界の主要コインシリーズを体系的に解説します。 |
| `/series/una-and-the-lion/` | `Una and the Lion（ウナ・アンド・ザ・ライオン）\| MINT COIN` | Una and the Lion の概要・歴史・種類・グレード・価格情報を日本語で解説。Royal Mint が発行する最も美しいコインのひとつ。 |
| `/grading/` | `グレーディング解説 \| MINT COIN` | NGC・PCGSによるコインのグレーディング（状態評価）の仕組み、シェルドンスケール、スラブコインの見方を解説します。 |

### キーワード戦略

#### 狙うキーワード（初期 Phase）

**高優先度（シリーズ名）:**
- Una and the Lion
- Three Graces コイン
- ゴシッククラウン
- Sovereign 金貨
- Britannia コイン
- Queen's Beasts
- Tudor Beasts

**中優先度（カテゴリ）:**
- アンティークコイン 種類
- モダンコイン 基礎知識
- NGC グレード 見方
- PCGS MS70
- eBay コイン 購入

**長尾（記事）:**
- Una and the Lion 価格
- Sovereign 1oz 金貨 相場
- NGC 提出 方法 日本
- アンティークコイン 保管 方法

#### 実装ルール

- H1 にはターゲットキーワードを含める
- description は 80〜120 文字で完結させる
- canonical を全ページに設定（BaseLayout で自動）
- OGP 画像は `/public/og/{slug}.png` に配置（将来）
- 内部リンクを積極的に貼る（特にシリーズページ間）
